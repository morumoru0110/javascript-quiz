## 概要
Javascriptを用いた超簡単クイズアプリです。</br>

## URL
https://murmuring-lowlands-35499.herokuapp.com/

## 作成理由
Javascript学習のアウトプットとして、もっとも簡単なクイズアプリを作りました。
プログラミングを題材に計5問、回答をクリックすると「正解/不正解」のアラートが発生し、全問終了すると結果が表示されます。
